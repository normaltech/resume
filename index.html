팀 소개 사이트 개발

2021.08.25 – 09. 07

프로젝트 개요:

웹개발을 공부하고 나서 처음 해보게 된 프로젝트
암호화폐 정보 게시, 팀원 소개 등의 팀을 소개하는 내용의 프론트엔드로만 만든 사이트

사용 기술:

  • HTML, CSS, JavaScript
  • AWS S3, CloudFront, Route53

프로젝트 진행 과정:

  • 첫번째로 맡게 된 개발 프로젝트로 지인을 통해 연결됨
  • 암호화폐 팀의 소개 사이트를 제작하는 일로 기존의 공부했던 HTML, CSS , JS와 기초적인 리액트사용법에 추가적으로 필요한 기능들의 공부와 패키지들을 추가하며 완성
  • 사이트가 오픈 하기로 한 시간까지 카운트 다운을 해주는 랜딩 페이지와 실제 소개 사이트 제작
  • 반응형 웹페이지로 개발요청이 있어 개별적인 공부를 통해 미디어 쿼리와 grid, flex속성을 통해 여러 화면 크기에 맞춰 css작업
  • 정적인 요소 이외의 것들이 추가적으로 필요를 느끼고 carousel과 동영상들과 email.js를 페이지에 추가하였습니다. carousel은 외부 패키지인 Flickity.js를 찾아서 팀의 암호화폐의 지표들과 구성팀들을 소개하는 글을 페이지에 넣었습니다.
  • 호스팅을 위해 유명했던 aws를 선택했고 s3라는 서비스로 호스팅했지만, 브라우저에서 해당 주소로 접속을 시도했을 때 제대로 연결이 안됐었고 그 이유를 찾아봤을 때 https가 적용이 안됐기 때문인 것을 알았습니다. https를 적용하기 위해 CloudFront로 s3를 호스팅하고 인증서를 발급받았고 실제로 쓰일 도메인을 route53에서 구입해서 문제없이 접속할 수 있게 저희 프로젝트를 호스팅을 하였습니다.
